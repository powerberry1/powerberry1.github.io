3. UPDATE THE SCENARIOS


* 수정 전
김씨는 인간컴퓨터상호작용이라는 강의를 들어왔다. 항상 그러듯이 교수님께서는 칠판에 필기가 많으시다. 김씨는 그것을 일일이 필기하기 싫기 때문에 교수님이 친판 한 바닥 필기가 끝나면 그것을 사진을 찍어왔다. 사진을 찍고 그 사진들이 어느 수업 때인지 정리해야 하지만 귀찮은 김씨는 수업이 끝나고 바로 밥을 먹으러 갔다. 그렇게 여러 수업을 듣고 난 뒤, 2주 뒤에 김씨는 중간고사 준비를 하려고 한다. 근데 찍은 사진들이 너무 많아서 김씨는 2주 전에 들었던 인간컴퓨터상호작용과목의 필기를 찾기가 힘들다. 그래서 그는 이 서비스를 사용한다. 도구창에서 사진 찾기를 누른다. 찾으려는 사진들의 날짜와, 시간을 선택하라는 창이 뜬다. 2018-10-15, 12:00~14:00을 선택하고 인간컴퓨터상호작용 폴더를 따로 만들어 둔 뒤, 선택한 사진들을 한 번에 인간컴퓨터상호작용폴더 안의 2019-10-15, 12:00~14:00 폴더에 저장을 한다.

* 수정 후
[예시 1]
1. 강의 들을 시 필기 대신 사진을 찍음
2. 인간과컴퓨터상호작용 시간에 찍은 사진을 찾고 싶음
3. 2018/9/1~2018/12/31 까지 날짜 설정
4. 월,수 요일 설정
5. 12:00~2:00 시간 설정
6. 설정한 날짜/요일/시간에 찍은 사진들을 날짜별로 모아줌

[예시 2]
1. 찍은 필기 사진들 중에서 RGB란 단어를 찾고 싶음
2. 단어 찾기 기능 선택
3. RGB 입력
4. RGB가 찍힌 사진들 체크해서 볼 수 있게 해줌

[예시 3]
1. 4/3~4/10 여행을 갔다옴
2. 그때 찍은 사진들만 모으고 싶음
3. 4/3~4/10 날짜 선택



4. UPDATE THE TASK MODEL


* 수정 전
Task Model 1: 강의별로 사진 모으기
1. 사진 모으기 버튼을 클릭한다.
2. 언제 찍은 사진을 모을지 정한다.(날짜 기간, 요일, 시간)
3. 모은 사진들의 폴더 이름을 지정한다.(인간과컴퓨터상호작용)
4. 인간과컴퓨터상호작용이란 큰 폴더 아래, 날짜별로 사진들이 따로 모아진다.

Task Model 2: 사진 합치기
1. 원하는 사진이 있는 폴더를 들어간다.
2. 사진 합치기 버튼을 클릭한다.
3. 예를 들어 칠판이 커서 나눠서 찍어야 했던 사진들이 합쳐져서 하나의 사진으로 변환된다.

* 수정 후
선행 : 사진첩에 있는 사진들 불러오기

Task Model 1: 강의별로 사진 모으기
1. 사진 모으기 버튼을 클릭한다.
2. 언제 찍은 사진을 모을지 정한다.(날짜 기간, 요일, 시간)
3. 모은 사진들의 폴더 이름을 지정한다.(인간과컴퓨터상호작용)
4. 인간과컴퓨터상호작용이란 큰 폴더 아래, 날짜별로 사진들이 따로 모아진다.

Task Model 2: 글자(용어 찾기)
1. 공부하고자 하는 교과목 폴더를 들어간다.(전체에서 찾고 싶으면 메인 페이지에서)
2. 글자 찾기 버튼을 누른다.
3. 찾고 싶은 글자를 입력한다.
4. 그 글자가 포함된 사진들을 찾아준다.
